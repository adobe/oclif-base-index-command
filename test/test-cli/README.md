# Usage
<!-- usage -->
```sh-session
$ npm install -g test-cli
$ test-cli COMMAND

$ test-cli (-v|--version|version)
test-cli/1.0.0 darwin-x64 node-v8.11.2
$ test-cli --help [COMMAND]
USAGE
  $ test-cli COMMAND
...
```
